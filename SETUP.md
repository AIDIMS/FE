# Hướng dẫn cài đặt dependencies

Để sử dụng các Shadcn UI components, bạn cần cài đặt các dependencies sau:

```bash
npm install @radix-ui/react-dialog @radix-ui/react-select
```

Sau khi cài đặt, các components sẽ hoạt động bình thường.

## Cấu trúc Layout

Dự án đã được cấu hình với:
- **Sidebar**: Navigation menu bên trái (desktop) và sheet (mobile)
- **Navbar**: Header với search, notifications, user menu
- **DashboardLayout**: Layout wrapper cho tất cả các trang quản lý

## Các trang hiện có

- `/dashboard` - Trang tổng quan
- `/patients` - Quản lý hồ sơ bệnh nhân
- `/records` - Quản lý hồ sơ (đang phát triển)
- `/settings` - Cài đặt hệ thống (đang phát triển)
